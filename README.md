# 🍔 Casa de Mamãe — Sistema de Pedidos Online

Sistema web desenvolvido para um negócio real, com foco em facilitar pedidos online e automatizar o atendimento via WhatsApp.

---

## 💡 Problema resolvido

Pequenos negócios enfrentam dificuldades em organizar pedidos feitos manualmente via WhatsApp, o que pode gerar erros, atrasos e perda de vendas.

O **Casa de Mamãe** resolve esse problema oferecendo um sistema simples e automatizado para gerenciamento de pedidos.

---

## ⚙️ Funcionalidades

- 🛒 Carrinho de compras
- 📦 Envio automático de pedidos via WhatsApp
- 📍 Validação de entrega por raio (3km)
- 📱 Interface responsiva (mobile first)
- ⚡ Estrutura preparada para PWA
